The “Difference Image” method for SNR calculation (5)
The “Difference Image” method for SNR calculation (5) Two identical MR images (Ssum and Sdif ) are acquired consecutively and a “Sum” and a “Difference” image are calculated as follow: $$ {S_{sum} =S_1+S_2 } $$ $$ { S_{dif} =S_1-S_2 } $$ A region of interest (ROI) is drawn on Ssum and the SNR is then calculated as the mean value inside the ROI, divided by the standard deviation inside the same ROI but applied to Sdif, : $$ { SNR = \frac{mean(ROI)_{S_{sum}} }{\sqrt{2} \cdot stdev(ROI)_{S_{dif}} } } $$
