SNR assessment using multiple image replicas
The “multiple replica” method is considered the gold-standard for SNR estimation. In the multiple replica method, k-space signals are acquired multiple times for the same slice and reconstructed using the same image reconstruction technique into a stack of equivalent image replicas which differ only in their noise content. The SNR is then assessed on a pixel-by-pixel basis as the ratio of the average (signal) and standard deviation (noise) of pixel values through the stack of image replicas. This method is useful for measuring the spatially-variant amplified noise in parallel imaging since image noise is calculated on a pixel-by-pixel basis.
